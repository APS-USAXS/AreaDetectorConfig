This directory contains file(s) to configure the 
EPICS AreaDetector ADCore software for this detector.

This directory is for the NDattributes file(s) and bad pixel files.

File-saving plugins have their own subdirectories:
* hdf5_plugin (start using 2015-02)
* nexus_plugin  (used until 2014-12)
